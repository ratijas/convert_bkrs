convert_bkrs
============

конвертация большого китайско-русско-китайского словаря в формат `.dictionary`.

![大БКРС](res/OtherResources/Images/icon.png)

## конвертация

```$ make```

сгенерирует 4 словаря:

- 大БКРС
- БРуКС
- 大БКРС+
- БРуКС+

"+" означает расширенные возможности поиска (см. ниже 'Extended Index').  чтобы
сконвертировать только один словарь, выполните одну из следующих комманд:

    $ make bkrs
    $ make bruks
    $ make bkrs-ei
    $ make bruks-ei

## Extended Index (расширенный индекс)

- БРуКС
    есть поддержка морфологии. слово можно найти по любым его формам. однако, из соображений экономии места, это касается только отдельных слов, но не словосочетаний.
- 大БКРС
    поиск по китайско-русскому направлению на стероидах (на примере 软件开发者):
    * 《软件》=> слова, которые начинаются на 《软件》
    * 《软件。》 => только 《软件》
    * если знаешь пиньинь, но не помнишь, как пишется, можно искать по пиньиню. пиньинь **нужно** разделять пробелами. вместо тонов можно использовать цифры. например:
        - ruan jian (*без тонов*)
        - ruǎn jiàn (*с диакритическими знаками*)
        - ruan3 jian4 (*с цифрами вместо диакритических знаков*)
        - ruan jian. (*точное совпадение*)

некоторые пользователи жаловались, что индекс получается очень уж раздутым и
занимает слишком много места, поэтому было решено сделать две версии: с
индексом и без.


## установка для текущего пользователя

`$ make install`

будет ругаться, если какая-то из версий не сконвертирована перед этим;  это
нормально.

## директории

* `./final`
	- здесь будут лежать готовые пакеты
* `./downloads`
	- сюда скачиваются базы в dsl формате
* `./img`
	- скриншоты приложения Dictionary.app для ознакомления
