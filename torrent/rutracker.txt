[b]Описание (взято с [url=http://bkrs.info]сайта словаря[/url])[/b]:

[b]大БКРС[/b] — открытый редактируемый большой китайско-русско-китайский словарь

основные принципы словаря:
[list]
[*] полнота - словарь включает в себя не только слова, но и устойчивые сочетания
[*] современность - словарь не включает в себя древнекитайский язык
[/list]

словарь сконвертиврован питоном из dsl файла, который можно скачать на сайте словаря
программу для конвертирования словарей dsl -> apple можно найти на [url=https://github.com/ratijas/convert_bkrs]github'е[/url]

[b]Доп. информация[/b]:
[spoiler="инструкция по установке"]
[list]
0) скачать
1) распаковать архивы со словарями (7z, две штуки), бесплатные программы для распаковки перечислены ниже
2) открыть новое окно Finder
3) выбрать меню "переход" -> "переход к папке…", или нажать [font="monospace"]⇧⌘G[/font]
4) ввести [font="monospace"]~/Library/Dictionaries[/font] и нажать enter
5) перетащить в эту папку распакованные ([size=11][i][color=green]см.[/color] пункт 2[/i][/size]) пакеты с расширением [b].dictionary[/b]
6) перезапустить приложение `Словарь`
7) зайти в настройки ([font="monospace"]⌘,[/font]), и в списке отметить галочку напротив 大БРКС, БРуКС
8) ...
9) PROFIT!!!

из замеченных [b]проблем с установкой[/b]:
окно предпросмотра не сразу начинает отображать слова из новоустановленных словарей. возможно, потребуется выход + вход в систему

[b]программы для работы с 7z-архивами[/b]:
[list]
[*] [url=http://www.macupdate.com/app/mac/20526/7zx]7zX[/url] — просто 7z
[*] [url=http://www.kekaosx.com/ru/]Keka[/url] — достойная замена стандартному архиватору Mac OS X
[*] [font="monospace"]$ brew install [url=http://p7zip.sourceforge.net/]p7zip[/url][/font] — для командной строки
[/list]
[/spoiler]

[spoiler="инструкция по использованию"]
[list]
[*] просто откройте приложение и наберите слово
[*] нажмите ⌃пробел (ctrl+пробел, для вызова spotlight) и наберите слово. статья из словаря появится в списке результатов. достаточно навести на неё курсор, чтобы увидеть перевод
[*] читая текст, наведите курсор на слово, или выделите кусок текста; далее нажмите ⌃⌘D ( ctrl+cmd+D ), или (для тех, кто с трекпадом) коснитесь сенсорной поверхоности тремя пальцами
[*] что бы убрать из списка результатов всё лишнее, добавьте в конце поиска точку. особенно полезно при поиске по пининю.
[*] цветной пиньинь. прям как на сайте, но ещё круче. с возможностью настроить по своему вкусу. достаточно зайти в настройки (⌘,)
[/list]

а также в версии с расширенным индексом:
[spoiler="БРуКС"]
есть поддержка морфологии. слово можно найти по любым его формам. однако, из соображений экономии места, это касается только отдельных слов, но не словосочетаний.
[/spoiler]
[spoiler="大БКРС"]
поиск по китайско-русскому направлению на стероидах (на примере 软件开发者):
[list]
[*] 《软件》=> слова, которые начинаются на 《软件》
[*] 《软件。》 => только 《软件》
[*] если знаешь пиньинь, но не помнишь, как пишется, можно искать по пиньиню. пиньинь [b]нужно[/b] разделять пробелами. вместо тонов можно использовать цифры. например:
 [list]
 [*] ruan jian ([i]без тонов[/i])
 [*] ruǎn jiàn ([i]с диакритическими знаками[/i])
 [*] ruan3 jian4 ([i]с цифрами вместо диакритических знаков[/i])
 [*] ruan jian. ([i]точное совпадение, про точку читай выше[/i])
 [/list]
[/list]
[/spoiler]
[/spoiler]
